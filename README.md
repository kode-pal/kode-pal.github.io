# kode-pal.github.io

A modern multi-page corporate IT consultancy website built with HTML5, CSS3, and vanilla JavaScript. Includes reusable components, responsive design, and scalable service pages.

## Overview

This project is a modern multi-page corporate website designed for IT consultancy services.

### Features

- Multi-page architecture (Home, Services, About, Contact)
- Reusable header & footer components
- Responsive design
- Modern corporate UI
- E-commerce & Android App Development service pages
- Clean folder structure
- Vanilla JavaScript (No frameworks)

### Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
